# Cloudette

To use these dotfiles on your computer, you first need to have a fresh minimal arch linux install.

#NOTE: YOU CAN EITHER USE THUNAR FILE MANAGER OR NAUTILUS FILE MANAGER.

#Commands

#1: sudo pacman -S hyprland hyprpaper waybar rofi swww nautilus kitty wlogout sddm starship cava

#OR:

#1: sudo pacman -S hyprland hyprpaper waybar rofi swww thunar kitty wlogout sddm starship cava 

#2: git clone https://github.com/Cloudette.git

#3: git clone https://aur.archlinux.org/yay.git

#4: cd yay

#5: makepkg -si

#6: Hyprland

#Now open the file manager which you installed by pressing SUPER (Windows) key + Q and then typing nautilus or thunar (Depends on which file manager you installed, if nautilus, type "nautilus" and if thunar, type "thunar")

#Press Ctrl + H

#Now locate the Cloudette folder. Select and open it, it has a folder called ".config". Select and copy it. Now, go to your home directory and paste it. When it will ask to replace, select "Yes".
